# All HTML

## Learning Objectives

This assignment addresses the following course learning outcomes:

- **LO1:** Use a basic vocabulary of interactive media to both give and respond to critique productively.
- **LO4:** Conceptualize a product, object, or experience for the web and realize it through coding.
- **LO9:** Combine your artistic creativity with technology related to the internet.

## Assignment Description

Create a series of 5 unique web pages which focus on one or two HTML tags each (no CSS whatsoever, even inline.) The idea of this assignment is to see how html tags look by default, and to use (or misuse) them in an inventive way.

## Assessable Tasks

- [ ] Create 5 unique HTML pages
- [ ] Each page focuses on 1-2 HTML tags
- [ ] Use **no CSS** (including no inline styles)
- [ ] Explore the default styling of HTML elements
- [ ] Use or creatively misuse HTML tags
- [ ] All pages are valid HTML

## Due Date

**Week 4: Feb 23**

## Deliverables

- 5 HTML files
- A README or index page explaining your approach
- Submit via GitHub repository

## Evaluation Criteria

- **Exploration:** Depth of exploration of HTML tags
- **Creativity:** Inventive use or misuse of HTML elements
- **Technical accuracy:** Valid HTML structure
- **Variety:** Diverse selection of HTML tags across pages
- **Understanding:** Demonstration of how HTML works without CSS
