# Expressive Text

## Learning Objectives

This assignment addresses the following course learning outcomes:

- **LO2:** Create compelling interactive experiences through more careful and inspired interpretation/translation of content (i.e. develop great design concepts)
- **LO6:** Evaluate how typography and its variables are applied to interactive systems to facilitate orientation, support usability and create consistency.
- **LO9:** Combine your artistic creativity with technology related to the internet.

## Assignment Description

Find or compose a piece of writing that can be experienced on a browser. The length of the text should be approximately one printed page. The text can be a poem, manifesto, quote, etc. Layout the text so the experience of reading changes based on scrolling through it.

## Assessable Tasks

- [ ] Select or compose a piece of text (approximately one printed page)
- [ ] Design a layout that responds to or enhances the text's meaning
- [ ] Create an experience that changes as the user scrolls
- [ ] Use HTML and CSS to implement your design
- [ ] Consider typography, spacing, and timing
- [ ] Ensure the design serves the content

## Due Date

**Week 5: Mar 2**

## Deliverables

- HTML file with your expressive text layout
- CSS file with your styling
- A brief written statement (in comments or README) explaining your design choices
- Submit via GitHub repository

## Evaluation Criteria

- **Concept:** How well the design interprets and enhances the text
- **Typography:** Thoughtful use of typographic variables
- **Interaction:** How the experience changes through scrolling
- **Technical execution:** Clean HTML/CSS code
- **Creativity:** Original approach to presenting text
- **Polish:** Attention to detail and refinement
