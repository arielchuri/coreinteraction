# Core Interaction Reading List

## Required Readings

These texts are referenced throughout the course and provide foundational knowledge for interaction design and web development.

### Books

1. **Casey Reas, Chandler McWilliams, and LUST** - *Form+Code in Design, Art, and Architecture*
   - Explores how code can be used as a creative medium
   - Examples of computational design processes
   - Relevant for understanding programming as design tool

2. **Kimberly Elam** - *Geometry of Design*
   - Visual analysis of design principles
   - Geometric foundations of composition
   - Applicable to grid systems and layout

3. **Armin Hofmann** - *Graphic Design Manual*
   - Swiss design principles
   - Systematic approach to visual design
   - Foundation for layout and composition

4. **Robert Bringhurst** - *The Elements of Typographic Style*
   - Comprehensive typography reference
   - Historical context and practical guidance
   - Essential for web typography (Week 7)

5. **Frank Chimero** - *The Shape of Design*
   - Philosophy of design practice
   - Creative process and decision-making
   - Available online: https://shapeofdesignbook.com/

6. **Leah Buley** - *The User Experience Team of One*
   - UX research and design methods
   - Practical approaches for solo practitioners
   - Relevant for Stories as Networks and Living Collection

7. **Laurel Schwulst (compiler)** - *Very Interactive Library*
   - Collection of interactive web precedents
   - Inspiration and technical examples
   - Available at: https://very-interactive-library.net/

8. **Paul Ford** - *What is Code?*
   - Comprehensive introduction to programming
   - Cultural and technical context
   - Available at: https://www.bloomberg.com/graphics/2015-paul-ford-what-is-code/

## Recommended Readings

### Web Design & Development

- **Ethan Marcotte** - *Responsive Web Design*
  - Foundation of responsive design principles
  - Mobile-first methodology

- **Jeremy Keith** - *HTML5 for Web Designers*
  - Practical guide to modern HTML
  - Semantic markup principles

- **Rachel Andrew** - *The New CSS Layout*
  - Modern CSS layout techniques
  - Flexbox and Grid

- **Eric Meyer** - *CSS: The Definitive Guide*
  - Comprehensive CSS reference
  - In-depth technical explanations

### Interaction Design

- **Don Norman** - *The Design of Everyday Things*
  - Foundational UX principles
  - Affordances and mental models

- **Steve Krug** - *Don't Make Me Think*
  - Usability principles
  - Web navigation and user experience

- **Scott McCloud** - *Understanding Comics*
  - Visual communication principles
  - Sequential narrative
  - Applicable to web design

### Digital Culture & Theory

- **Nicholas Carr** - *The Shallows: What the Internet Is Doing to Our Brains*
  - Impact of digital media on cognition
  - Reading and attention in digital age

- **Lev Manovich** - *The Language of New Media*
  - Theory of digital media
  - Database logic and interface culture

- **Janet Murray** - *Hamlet on the Holodeck*
  - Interactive narrative
  - Digital storytelling

### Typography

- **Ellen Lupton** - *Thinking with Type*
  - Visual guide to typography
  - Practical exercises and examples

- **Matthew Butterick** - *Butterick's Practical Typography*
  - Web typography guide
  - Available online: https://practicaltypography.com/

## Building Your Own Library

As you progress in interaction design, consider building your own library of references:

1. **Bookmark useful resources** - Save tutorials, documentation, and inspiration
2. **Take notes** - Annotate readings with your own observations
3. **Create examples** - Build small demos based on techniques you read about
4. **Share discoveries** - Contribute interesting finds to class discussions

## Staying Current

Web technologies evolve rapidly. Follow these practices:

- Subscribe to web design/dev newsletters
- Follow designers and developers on social media
- Attend local meetups or conferences (when possible)
- Experiment with new techniques and tools
- Read release notes and changelogs for major tools

## Citation

When referencing these works in your assignments or Learning Portfolio:

- Include author names, titles, and publication years
- Provide URLs for online resources
- Give credit for code examples and techniques
- Maintain a bibliography for major projects
