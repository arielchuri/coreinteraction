### 8. Review File Naming Best Practices
Understanding proper file naming is crucial for web development!

**Why it matters:**
- Web servers are case-sensitive
- Spaces break URLs
- Consistency prevents errors

**Key rules:**
- ✅ Always use lowercase
- ✅ No spaces (use hyphens or underscores)
- ✅ Be descriptive
- ❌ Avoid special characters

**Reference:** [File Naming Cheat Sheet](https://github.com/arielchuri/coreinteraction/blob/main/.storage/class-notes/file-naming-cheatsheet.md) (detailed guide available in class materials)
